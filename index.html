An HTML element is a type of HTML document component, one of several types of HTML nodes.

An HTML element opening tag is declared with a word or letter in angle brackets. After this declaration a text can be typed

An HTML element closing tag is declared with a backslash and a word or letter in angle brackets. After this declaration the element is closed and nomore text can be written within the element

The id attribute specifies a unique id for an HTML element. The value of the id attribute must be unique within the HTML document.
The id attribute is used to point to a specific style declaration in a style sheet. It is also used by JavaScript to access and manipulate the element with the specific id.

The HTML class attribute specifies one or more class names for an element. Classes are used by CSS and JavaScript to select and access specific elements. The class attribute can be used on any HTML element. The class name is case sensitive. Different HTML elements can point to the same class name.
